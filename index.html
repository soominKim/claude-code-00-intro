01 — OVERVIEW

큰 그림부터

Claude는 사람으로 비유하면 엄청 똑똑한 비서입니다. 하지만 혼자서는 당신의 Notion, Figma, 엑셀 파일을 보지도 못하고, 자주 쓰는 당신만의 루틴을 기억하지도 못합니다.

그래서 필요한 게 확장 요소들입니다.

  • 01내 파일을 다룰 수 있게 해주는 스킬(Skills)
  • 02특정 분야에 능한 에이전트(Subagents)
  • 03외부 도구로 들어가는 열쇠인 MCP
  • 04반복 요청을 줄여주는 슬래시 커맨드(Slash Commands)
  • 05대화창에서 실행되는 앱(Apps)
  • 06특정 순간에 자동 작동하는 훅(Hooks)
그리고 이것들을 한 번에 묶어 배포하는 플러그인(Plugins)도 있습니다. 이 사이트는 그 모든 확장 요소를 쇼핑하듯 골라 담아 설치할 수 있게 만들어졌습니다.
02 — CONCEPTS

하나씩 비유로

01

Skills

필요할 때만 꺼내 쓰는 전문 과외 선생

평소엔 숨어 있다가 특정 파일이나 요청이 들어오면 자동으로 깨어나 도와주는 전문 지식 묶음. 엑셀 파일이 들어오면 '엑셀 선생님'이, PDF가 들어오면 'PDF 선생님'이 들어옵니다.

예시

가계부 엑셀을 첨부하고 '이번 달 지출 정리해줘'라고 부탁하면, XLSX 스킬이 자동으로 깨어나 시트를 읽고 계산합니다.

비유로 다시

학교에서 수학 시간엔 수학 선생님이, 영어 시간엔 영어 선생님이 들어오는 것처럼, Claude도 상황에 맞는 스킬이 자동으로 들어와 가르쳐 줍니다.

02

Subagents

같은 팀의 전문 동료

Claude 혼자 다 하지 않고, 어떤 종류의 일은 그 분야에 최적화된 '동료'에게 맡깁니다. 탐색 담당, 계획 담당, 리뷰 담당 같이 역할이 나뉘어 있어요.

예시

'이 프로젝트의 인증 관련 파일 전부 찾아줘'라고 부탁하면, Claude가 Explore 동료에게 일을 넘겨 뒤에서 뒤지게 합니다. 대화창엔 결과만 돌아옵니다.

비유로 다시

프로젝트 매니저가 혼자 다 하지 않고 디자이너·개발자·QA에게 맡기듯, Claude도 상황에 맞는 에이전트에게 일을 넘깁니다.

03

Plugins

관련된 것들을 한 박스에 넣은 선물 세트

서로 관련된 스킬·에이전트·단축키·자동화를 하나의 박스로 묶어 배포하는 패키지. 개별로 설치하는 대신 박스 하나로 여러 기능을 갖춥니다.

예시

'오피스 작업 세트' 플러그인을 설치하면 엑셀·워드·PPT·PDF 스킬이 한꺼번에 활성화됩니다.

비유로 다시

이사 갈 때 식기 낱개로 사는 대신 '6인 세트' 한 박스로 사는 것과 비슷합니다.

04

MCP Servers

외부 서비스로 들어가는 열쇠 꾸러미

Claude가 Notion·Figma·GitHub·구글 드라이브 같은 외부 서비스를 '직접 보고 다룰 수 있게' 해 주는 열쇠. 열쇠가 없으면 Claude는 바깥 세상을 볼 수 없습니다.

예시

Notion MCP를 연결하면 '지난 주 회의록에서 할 일 목록 뽑아줘'라고 부탁할 수 있게 됩니다.

비유로 다시

집에 들어가려면 열쇠가 있어야 하듯, Claude가 내 Notion 워크스페이스에 들어가려면 MCP 열쇠가 필요합니다.

05

Slash Commands

자주 쓰는 긴 요청을 한 줄로 줄인 단축키

매번 같은 긴 설명을 하지 않고 /리뷰, /초기화 한 줄로 정해진 작업을 호출하는 명령어. 스킬의 가벼운 버전이라고 보면 됩니다.

예시

/review라고 치면, Claude가 현재 변경사항 전체를 리뷰해 코멘트를 달아 줍니다.

비유로 다시

키보드의 Ctrl+C, Ctrl+V처럼 Claude의 단축키. 몇 글자 외워두면 반복 업무가 반으로 줄어요.

06

Apps

Claude 안에서 돌아가는 작은 앱

Claude.ai 대화창 안에서 바로 실행되는 미니 프로그램. 브라우저 탭을 오가지 않고 그 자리에서 디자인을 그리거나 문서를 편집할 수 있게 해 줍니다.

예시

Claude Design 앱에서 '회사 소개 원페이저 만들어줘'라고 부탁하면 바로 디자인된 결과물이 나옵니다.

비유로 다시

카카오톡 안에 있는 '미니 앱'처럼, Claude 대화 안에 있는 작은 앱들입니다.

07

Hooks

특정 순간에 자동으로 반응하는 센서

'이런 일이 벌어지면 이걸 자동으로 해라'를 미리 걸어두는 자동 반응 규칙. 파일을 고친 직후 포맷팅을 자동 실행하거나, 위험한 명령을 막을 수 있습니다.

예시

'파일 수정 후 자동 포맷팅' 훅을 걸어두면, Claude가 코드를 고치자마자 prettier가 자동으로 돌아가 스타일을 정리합니다.

비유로 다시

동작 감지 센서가 달린 자동문 같아요. 무언가 지나가면 자동으로 열리듯, 특정 이벤트가 생기면 정해둔 동작이 바로 실행됩니다.

03 — IN PRACTICE

실전 예시 · 바로 써보기

이론은 이 정도면 충분합니다. 실제로 어떻게 설치하고, 어떤 식으로 요청하면 되는지 가장 먼저 해야 할 한 가지부터 자주 쓰이는 Skills·MCP 순으로 보여드릴게요.

가장 먼저 · 필수

CLAUDE.md — 프로젝트 규칙 적어두기

Claude Code 공식 문서가 초보자에게 가장 먼저 권하는 것. 프로젝트 루트에 CLAUDE.md 파일 하나 만들어 두면, 모든 세션이 시작될 때 자동으로 읽힙니다. "항상 이렇게 해라" 류의 규칙을 여기 적어두세요.

# 프로젝트 루트에 CLAUDE.md 생성 — Claude Code에서 /init # 코드베이스를 훑어보고 CLAUDE.md 초안을 자동 작성합니다
초보자 팁 · 200줄 넘기지 말 것
CLAUDE.md는 짧고 핵심만. 길어지면 Claude가 오히려 규칙을 놓쳐요. 상세한 참고 자료는 Skill로 분리하는 게 낫습니다.
Skills · 권장

document-skills 플러그인 설치

PDF · DOCX · XLSX · PPTX 네 개의 스킬이 한 번에 들어오는 공식 플러그인. 문서 다루는 일이 조금이라도 있다면 이것부터 깔아두세요.

설치 없이 바로 쓸 수 있는 기본 스킬
Claude Code엔 /simplify(코드 단순화) · /batch(반복 작업 일괄 처리) · /debug(디버깅 도우미) 같은 번들 스킬이 이미 깔려 있습니다. 따로 설치 안 해도 바로 /로 호출 가능.
# Claude Code에서 순서대로 실행 /plugin marketplace add https://github.com/anthropics/skills /plugin install document-skills@anthropic-agent-skills /plugin list
마켓플레이스 (anthropic-agent-skills) └─ 플러그인 (document-skills) ← 설치 단위 ├─ 스킬: pdf ├─ 스킬: docx ├─ 스킬: xlsx └─ 스킬: pptx
구조 이해하기
마켓플레이스는 '앱스토어', 플러그인은 '앱 묶음', 스킬은 '앱 안의 기능' — 플러그인 하나만 깔면 스킬 네 개가 한꺼번에 활성화됩니다.
PDF Skill · 프롬프트 예시

이런 식으로 부탁하면 됩니다

스킬은 따로 호출하지 않아도 자동으로 깨어납니다. PDF를 첨부하고 자연어로 요청만 하면 돼요.

KO · 이미지 추출 이 제안서 PDF에서 아키텍처 다이어그램이랑 화면 목업 이미지들 전부 추출해서 별도 폴더에 저장해줘. 파일명은 페이지+설명으로.
EN · Same request Use the PDF skill to extract all architecture diagrams and UI mockups from this proposal PDF. Save to a separate folder with filenames as page_number + description.
KO · 여러 PDF 병합 ~/Documents/CHI2026 폴더에 있는 논문 PDF 8개를 저자 알파벳순으로 정렬해서 하나로 합쳐줘. 목차 페이지도 앞에 만들어주고.
KO · 챕터별 분할 이 200페이지짜리 PDF를 챕터별로 나눠서 별도 파일로 만들어줘.
KO · 표 추출 → 엑셀 이 PDF 23–27페이지에 있는 비용 산출표들 다 뽑아서 하나의 엑셀 파일로 합쳐줘. 시트는 페이지별로 나눠서.
MCP · 권장

Claude.ai의 Connectors 활용

Claude Code에서 MCP 서버를 직접 설정하는 것도 가능하지만, 가장 쉬운 시작은 Claude.ai의 Connectors 메뉴입니다. Notion · Gmail · Google Calendar · Google Drive · Slack · Figma · Notion · Microsoft Learn 같은 자주 쓰는 서비스가 클릭 몇 번으로 연결돼요.

연결해 두면 좋은 것들
· Notion — 회의록·프로젝트 위키에서 정보 찾고 업데이트
· Gmail · Calendar — 메일 초안 작성, 일정 잡기
· Google Drive — Drive 안 문서 검색·요약
· Figma — 디자인 파일에서 스펙·스크린샷 가져오기
· Slack — 채널 검색, 메시지 초안·예약 발송
예시 · Notion Connector 지난주 팀 회의록에서 내 이름이 들어간 할 일 전부 뽑아서 우선순위별로 정리해줘.
예시 · Figma Connector 이 Figma 링크의 화면을 그대로 구현할 React 컴포넌트 코드 초안 작성해줘.
04 — PRODUCTS

Claude 제품 라인업

Anthropic은 용도에 따라 여러 형태의 Claude를 만듭니다. 이 사이트에서 담는 기능은 대부분 Claude Code와 Claude.ai 두 군데에 설치됩니다. 나머지는 제품 자체가 하나의 카탈로그 아이템이에요.

Claude.ai 웹 · 데스크탑 · 모바일

가장 흔하게 만나는 Claude. 브라우저의 claude.ai, 맥·윈도우 데스크탑 앱, iOS·안드로이드 모바일 앱이 모두 같은 계정으로 연결돼 한 대화를 이어 갑니다.

Claude Code CLI · 데스크탑 · IDE 확장

개발자를 위한 Claude. 터미널(CLI)이 가장 강력하고, 별도 데스크탑 앱과 VS Code·JetBrains 확장도 있습니다. Skills·MCP·Hooks 같은 고급 확장이 전부 가능합니다.

Claude Cowork 별개 데스크탑 앱

Claude Code의 비개발자 버전. 에이전트가 내 컴퓨터의 파일·이메일·엑셀 같은 앱을 자동으로 오가며 일을 끝내 줍니다.

Claude Design 비주얼 제작 앱

프로토타입·슬라이드·원페이저·목업을 말로만 시켜 만드는 Anthropic Labs의 신작.

Claude API · Agent SDK

Claude를 자기 서비스에 붙이고 싶을 때 쓰는 프로그래밍 인터페이스. 이 사이트의 주 대상은 아니지만, 개발자라면 이쪽으로도 거의 모든 기능을 연결할 수 있습니다.
